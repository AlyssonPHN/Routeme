# Routeme
AppTeste

Route me usa o google maps:
- Mostra sua posição atual;
- Procura endereço com auto complete;
- Traça a rota

Utilizado Designer Pattner, MVVM, Injeção de dependência (Dagger-Hilt) e Teste Unitário (Mockito)
